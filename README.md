# Gestion-de-temp-rature
README - Système de Monitoring de Température (IoT)

Ce projet est un contrôleur intelligent permettant de : - Configurer des
seuils de température. - Lire et sauvegarder la configuration. - Simuler
des mesures de température (aléatoires ou fixes). - Détecter et
classifier les alertes en 3 niveaux. - Enregistrer les mesures dans un
fichier journal. - Générer des statistiques complètes. - Convertir des
températures (Celsius, Kelvin, Fahrenheit). - Convertir les températures
enregistrées dans le journal vers d’autres unités. - Exécuter un mode de
surveillance continue.

Fichiers générés automatiquement

-   config.txt : sauvegarde des paramètres de configuration.
-   journalisation.txt : journal des mesures.
-   rapport_journalier.txt : rapport statistique.
-   journalisation_converti.txt : journal converti en unités choisies.

Compilation

    gcc monitoring.c -o monitoring

Exécution

    ./monitoring
